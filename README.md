# 洛克王国双组别种公计算器

一个用于计算洛克王国中最佳双组别宠物组合的工具，帮助玩家找出能够精确覆盖全部组别且星光值最高的种公宠物组合。

## 功能简介

洛克王国中，部分宠物拥有双组别属性。本工具通过精确覆盖算法，自动找出所有恰好7只宠物的组合，使其能够覆盖全部14个组别（两栖、动物、大地、天空、妖精、巨灵、拟人、昆虫、机械、植物、海洋、软体、魔力、龙），且每个组别只被覆盖一次，并按星光值从高到低排序。

## 文件说明

| 文件 | 说明 |
|------|------|
| `find_combos_score.py` | 核心计算脚本 |
| `s2全部可做种公双组别宠物.xlsx` | S2赛季所有可做种公的双组别宠物大全 |
| `我的最佳.xlsx` | 用户自定义宠物列表（从大全中筛选后） |
| `results.txt` | 计算结果输出文件 |

## 使用方法

### 第一步：准备宠物列表

1. 打开 `s2全部可做种公双组别宠物.xlsx` 查看所有可用的双组别宠物
2. 根据自己的需求，删除难抓或不想要的宠物
3. 将筛选后的列表保存为 `我的最佳.xlsx`

### 第二步：运行计算脚本

```bash
# 使用默认文件（我的最佳.xlsx）
python find_combos_score.py

# 或指定其他Excel文件
python find_combos_score.py 你的宠物列表.xlsx
```

### Excel文件格式要求

第一行为表头，之后每行包含：

| 列 | 内容 | 说明 |
|----|------|------|
| 第1列 | 宠物名 | 宠物名称 |
| 第2列 | 组别1 | 第一个组别 |
| 第3列 | 组别2 | 第二个组别 |
| 第4列 | 星光值 | 宠物星光值（可选，默认0） |

## 输出结果

计算完成后，结果会保存到 `results.txt` 文件中，包含：

- 输入文件名
- 覆盖的组别列表
- 所有有效组合（按星光值从高到低排序）

## 技术原理

本工具采用**回溯算法**结合**精确覆盖**策略：

1. **精确覆盖**：每只宠物覆盖2个组别，7只宠物恰好覆盖14个组别，无重叠
2. **字典序优化**：始终选择字典序最小的未覆盖组别作为分支点，避免重复计算相同组合
3. **剪枝优化**：当剩余组别数量超过剩余宠物槽位能覆盖的数量时，提前终止该分支

## 注意事项

1. Excel文件必须为 `.xlsx` 格式
2. 请确保宠物列表能够覆盖全部14个组别，否则可能无法找到有效组合
3. 组别名称必须与现有数据一致（两栖、动物、大地、天空、妖精、巨灵、拟人、昆虫、机械、植物、海洋、软体、魔力、龙）
